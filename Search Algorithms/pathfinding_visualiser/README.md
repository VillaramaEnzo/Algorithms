# BFS & DFS Pathfinding Visualizer

A React + TypeScript visualization tool for comparing BFS and DFS pathfinding algorithms.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (usually http://localhost:5173)

## Project Structure

```
bfs-dfs-visualizer/
```
